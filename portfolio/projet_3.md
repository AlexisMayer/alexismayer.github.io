# Description du projet 

Cette étude pour le groupe ACCOR s'inscrit dans un questionnement plus large:
La rentabilité des hôtels ACCOR peut-elle être améliorée par la  satisfaction de
la clientèle et comment? Il est généralement admis que l'expérience  client est 
le moteur des ventes pour les hôtels. Cependant, cette hypothèse n’a pas été 
démontrée empiriquement. L'originalité de l'étude est d'analyser l'évolution de
satisfaction client et de performance hôtelière (estimé ici par le nombre de réservations) 
dans le temps, en utilisant les séries chronologiques des trois dernières années,
afin d’explorer cette hypothèse.
